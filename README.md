gpioPy
======

About
-----

Orange PI and Orange PI Zero GPIO command written in Python

Usage
-----

Modify **gpio.py** script to **Orange Pi** or **Orange Pi Zero** boards.

```
# Change Pinout board (PINOUT_OPI / PINOUT_OPI_ZERO)
PINOUT = PINOUT_OPI
```

Run the help scripts.

```
./gpio.py help
./board-led.py help
```
